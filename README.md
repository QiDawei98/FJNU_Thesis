# FJNU_Thesis
这个项目来自一次噩梦般的经历，我本人的word版硕士论文所包含的大量的数学公式突然一夜之间全部乱码了。最有意思的是，在完全不做修改的情况下反复打开word乱码的方式还不太一样。

除了页眉的部分细节以外，本模板完整复刻了 师大研【2019】7号 关于印发《福建师范大学研究生学位论文基本要求与书写格式（修订）》的通知 一文中的规定。其中部分不清楚的地方。我尽可能地从视觉上模仿了张守鑫同学《磷酸锆多孔材料的结构调控及对铀（VI）吸附性能研究》和刘国粮同学《铜掺杂铅磷灰石及钐掺杂钛酸锶的电子结构计算》两篇论文在Word中的实际效果。

随着我本人硕士学位论文不断完善，本模板将会逐步更新并添加更多的细节。如果有任何的意见和建议，欢迎反馈。

本项目的代码完全由AI生成，并经过了人工审阅。
